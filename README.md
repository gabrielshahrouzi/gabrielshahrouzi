# Gabriel Shahrouzi

Building AI + systems tools | CS PhD @ UMass Lowell

I build software that compresses repetitive workflows into simple, high-leverage tools — from browser automation for university systems to low-level Linux kernel contributions.

## Projects

- **JobHawk Autofill** – Chrome extension that turns repetitive JobHawk applications into a one-click workflow.  
  Saves profile data once, then fills application fields including availability, resume selection, work authorization, major, and campus work status.  
  [View on Chrome Web Store](https://chromewebstore.google.com/detail/application-autofill-for/jhmlodpmidoabldkbjelcimabknpegej)

- **JobHawk Auto Timesheet** – Chrome extension that reduces timesheet entry friction on UML JobHawk.  
  Tracks work sessions, infers start/end times, saves notes locally, and minimizes repeated manual input.  
  [View on Chrome Web Store](https://chromewebstore.google.com/detail/jobhawk-auto-timesheet/kcpggheldbnmeafighgcpeamnkmohooi)

- **Opportunity Finder** – web app for discovering, aggregating, and organizing opportunities.  
  https://opporunity-finder.vercel.app/

- **LeetCode Trainer** – structured coding interview practice tool for deliberate problem training.  
  https://leetcode-trainer.vercel.app/

- **UML Aggregator** – web app for aggregating and organizing UML-related information.  
  https://uml-aggregator.vercel.app/
